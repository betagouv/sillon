# Puis-je ouvrir mon code ?

Oui, sauf dans quelques rares cas comprenant l'un des points suivants :

- Du contenu secret défense ;
- Des risques pour la cybersécurité ;
- Des risques d'infraction.

:::info Licence
Utilisez une licence permissive pour votre code. Plus d'infos dans [la page legal/licence](legal/license.mdx).

Consultez le [Guide juridique interactif sur la publication des codes sources de l'administration](https://code.gouv.fr/guides/juridique/).
:::
