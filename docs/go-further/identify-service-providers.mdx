# Identifier des prestataires de services

En tant qu'administration ou collectivité territoriale vous avez accès [au marché multi-éditeurs UGAP](https://www.ugap.fr/) (union des groupements d'achats publics). Cette centrale d'achat public :

- couvre divers domaines (informatique, santé...) ;
- permet en fonction de vos besoins d'identifier les prestataires adéquats ;
- existe afin de simplifier les procédures d'achat.
