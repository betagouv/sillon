# Est-ce que votre projet est inédit ?

Il est important de prendre quelques heures voire quelques jours pour sonder le marché afin d'être sûr qu'une solution n'existe pas déjà. Il faut :

- Être vigilant à ce qui existe dans sa propre entité et dans les autres administrations publiques. Si une telle solution existe, éviter de faire doublon et rentrer directement en communication avec leur équipe produit pour réutiliser le leur ou essayer de mutualiser le besoin ;
- Dans le cas où c'est un service privé, jauger si ça vaut vraiment le coup de développer une solution en interne comme expliqué dans [les coûts liés au temps](project-start/time-and-costs.mdx). C'est un arbitrage très souvent lié à vos spécificités métiers non présentes dans l'outil.

:::tip Aide à la recherche
**Pour les projets publics**, en plus du traditionnel moteur de recherche vous pouvez vous appuyer sur :

- [Établi, un annuaire qui référence les initiatives publiques numériques](https://etabli.incubateur.net/) _(possibilité de rechercher par besoin, en utilisant soit la recherche classique, soit l'assistant virtuel)_ ;
- [L'annuaire de service-public.fr](https://lannuaire.service-public.fr/) _(recherche classique)_.

:::
